Aumenta la cantidad de un producto seleccionado en 1. Si el producto no está seleccionado, no hace nada. Se utiliza para ajustar la cantidad de productos en la vista de selección de productos, cuando ya están seleccionados.

void aumentarCantidad(Producto p){
    if(_productosSeleccionados.containsKey(p)){
      _productosSeleccionados[p] = _productosSeleccionados[p]! + 1;
      notifyListeners();
    }
